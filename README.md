# Interactive Bar Chart By Vanilla JS

## Getting Started

```bash
git clone https://github.com/weeeeey/interactive-chart
```

run the development server:

### VSCode

Install the Live Server Extension

### Node.js

```bash
npm install -g http-server
```

```bash
http-server
```
